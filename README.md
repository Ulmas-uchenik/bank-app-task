# Spring Security

Сделал я значит проект с регистрацией. У меня есть следующие виды регистрации:
- Basic Auth (аутентификации хранится в header-ах запроса, и ее нельзя ни как от-туда удалить => разлогиниться не получится)
- Form Auth (данные хранятся на сервере в Spring контекст в сессии пользователя)
- Jwt Auth (авторизация через подписанные access token)
- OAuth 2 (Вход через google почту, после чего ваши данные сохраняются в мою бд, а вам в cookie выставляется Access Token)


Urls: 
- Basic: /basic
- form: /form
- jwt: смотреть в LoginController
- google: /login/google


JwtService 
- тут у меня происходит вся логиа с токенами. Их отклонение и все остальное.
